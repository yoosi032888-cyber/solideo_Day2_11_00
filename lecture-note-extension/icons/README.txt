아이콘 생성 방법
==================

방법 1: generate-icons.html 사용 (추천)
----------------------------------------
1. generate-icons.html 파일을 브라우저에서 엽니다.
2. 자동으로 아이콘이 생성됩니다.
3. 각 아이콘을 우클릭 → "이미지를 다른 이름으로 저장"
4. 파일 이름을 각각 icon16.png, icon48.png, icon128.png로 저장
5. 이 폴더(icons)에 저장합니다.

방법 2: 온라인 도구 사용
------------------------
1. https://www.favicon-generator.org/ 방문
2. 원하는 이미지 업로드 또는 생성
3. 16x16, 48x48, 128x128 크기로 다운로드
4. 파일 이름을 icon16.png, icon48.png, icon128.png로 변경
5. 이 폴더에 저장

방법 3: 직접 제작
-----------------
Photoshop, GIMP, Canva 등의 도구를 사용하여
16x16, 48x48, 128x128 크기의 PNG 이미지를 제작하고
이 폴더에 저장합니다.

참고:
- 아이콘은 심플하고 명확한 디자인이 좋습니다.
- 배경은 투명하거나 단색으로 하는 것을 권장합니다.
- 🎓 (졸업모자) 이모지를 활용하면 좋습니다.
